# Christina's World (个人主页)

这里是 Christina 的个人网站源代码仓库。

👋 **[点击这里访问我的网站 (Live Site)](https://www.christinayxr.com)**

## 🏠 关于本项目
这是我的个人数字花园，用于记录和分享我的创作与生活。网站主要包含以下板块：
* **📚 小说**：我的科幻与现实主义题材连载小说。
* **📷 摄影**：使用 奥林巴斯 拍摄的街头与风景。
* **📝 笔记**：关于全栈开发、Jekyll 与云计算的学习路径。

## 🛠️ 技术栈 (Tech Stack)

* **核心引擎**: [Jekyll](https://jekyllrb.com/)
